# 🇮🇳 Indic-GPT — Multilingual AI Assistant for Indian Languages

Indic-GPT is a multilingual AI assistant designed to understand and respond in **Indian languages**.  
It allows users to chat naturally in Hindi, Marathi, Tamil, Telugu, Hinglish, and more — making AI more accessible for regional users.

Built during **Intellithon-26**, this project focuses on breaking language barriers using modern LLM APIs and a clean chat interface.

---

# ✨ Features

* 🌐 Multilingual chat (Indic languages)
* 💬 Conversational AI interface
* 🔁 Same-language responses
* 🧠 Context-aware answers
* ⚡ Fast AI responses
* 🎨 Clean UI with React + Tailwind
* 📱 Fully responsive design

---

# 🧠 How It Works

1. User types message in any Indic language  
2. Input is processed in the frontend  
3. LLM API generates contextual response  
4. Response returned in same language  
5. UI displays formatted AI reply  

```
User → React UI → LLM API → Response → UI
```

# 🚀 Getting Started

## 1. Clone the repository

```bash
git clone https://github.com/lucky-KRMU/Intellithon-26-Indic-GPT.git
cd Intellithon-26-Indic-GPT
```

---

## 2. Install dependencies

```bash
npm install
```

---

## 3. Setup Environment Variables

Create `.env` file in root:

```
VITE_API_KEY=your_api_key_here
```

---

## 4. Run the project

```bash
npm run dev
```
